How does it work

The Bot allows you to be notified as soon as a product arrives

Select your filters

Go to vinted and search using the desired filters. All new articles coming to appear on the page will be notified.

Use Discord to start the Bot

It is possible to use the /sub command on Discord followed by the search url to start the bot.

Send us the link corresponding to your search

If you don't have access to Discord or don't want to use the previous method, you can send us the link.

You will be added as soon as possible to a channel corresponding to your request and you will receive notifications as soon as an item arrives

Receive your notifications

Once the url has been added to the channel, you will start receiving notifications as soon as an article is published.
