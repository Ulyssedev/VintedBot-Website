    Voici quelques fonctionnalités du Vinted Bot

    Suivi Vinted personnalisé

    Avec la commande /sub, vous pouvez vous abonner à un mot-clé spécifique avec les filtres souhaités. Lorsqu'une nouvelle annonce est publiée avec ce mot-clé et/ou avec les filtres sélectionnés, vous recevrez une notification dans un salon Discord.

    Booster de vues

    Avec la commande /views, vous pouvez augmenter les vues d'un article. Le bot ajoutera automatiquement le nombre de vues souhaité à l'article.

    Informations utilisateur avancées

    En cliquant sur le bouton "Informations vendeur" sur n'importe quelle notification, vous recevrez un message contenant une note de confiance et d'autres informations sur l'utilisateur.

    Analyse de recherches

    Avec la commande /analytics, vous pouvez obtenir un rapport détaillé sur le prix des articles de votre recherche.
